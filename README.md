# 🎹 Treinador de Notas Musicais QWERTY

O **Treinador de Notas Musicais QWERTY** é uma aplicação web interativa criada para auxiliar o estudo de notas musicais a partir do teclado do computador. A ferramenta associa teclas do teclado QWERTY a nomes de notas e a conjuntos sonoros reproduzidos em um piano virtual, registrando a performance em um campo de texto que funciona como uma espécie de **máquina de escrever musical**.

Além do treino de inserção de notas, o sistema oferece recursos de prática como metrônomo, seleção de timbre, controle de volume, reverb, sustentação e modo de alto contraste.

## ✨ Funcionalidades

- Piano virtual interativo exibido na tela
- Associação de teclas do teclado QWERTY a notas musicais
- Reprodução sonora com diferentes timbres
- Registro automático das notas tocadas em um campo textual
- Organização do texto em compassos numerados
- Metrônomo integrado com:
  - controle de liga/desliga
  - ajuste de BPM
  - controle de volume
- Ajustes de áudio:
  - volume principal
  - reverb
  - sustentação
- Botão para limpar o texto da performance
- Botão para copiar a performance formatada
- Inserção automática de citação musical ao copiar
- Modo de alto contraste para acessibilidade
- Opção para ocultar e mostrar controles
- Destaque visual das teclas tocadas no piano

## 🎯 Objetivo do Projeto

O projeto foi desenvolvido para apoiar atividades de percepção, reconhecimento e prática de notas musicais usando o teclado do computador como interface. A proposta é unir treino auditivo, resposta motora e registro simbólico em um ambiente simples e acessível, útil para contextos educativos, oficinas, aulas de música e experimentação criativa.

## ⚙️ Como funciona

Cada tecla do teclado ativa um grupo de notas associado a um nome musical:

- **A** → Lá
- **B** → Si
- **C** → Dó
- **D** → Ré
- **E** → Mi
- **F** → Fá
- **G** → Sol

Ao pressionar uma dessas teclas:

1. o sistema toca um conjunto de notas em diferentes oitavas;
2. destaca visualmente as teclas correspondentes no piano virtual;
3. adiciona o nome da nota ao campo de texto;
4. organiza a escrita em blocos de quatro notas por compasso.

## ▶️ Como usar

1. Abra o arquivo HTML em um navegador moderno.
2. Digite seu nome no campo de identificação, se desejar.
3. Clique em qualquer lugar da página para inicializar o áudio.
4. Pressione as teclas:
   - **A**
   - **B**
   - **C**
   - **D**
   - **E**
   - **F**
   - **G**
5. Observe:
   - o som reproduzido;
   - o destaque no piano virtual;
   - o texto sendo construído na área da máquina de escrever musical.
6. Ajuste o metrônomo e os parâmetros sonoros conforme necessário.
7. Use **Copiar sua performance!** para exportar o resultado.

## 🧩 Estrutura da Interface

A interface está dividida em blocos principais:

### 1. Cabeçalho
Apresenta o título do projeto e botões de acessibilidade e visualização.

### 2. Área de entrada e máquina de escrever musical
Inclui:
- nome do usuário;
- metrônomo;
- visor do texto musical;
- botões de limpar e copiar.

### 3. Painel de controles
Permite ajustar:
- timbre;
- volume;
- reverb;
- sustentação.

### 4. Piano virtual
Mostra visualmente as teclas correspondentes às notas.

## 🎨 Interface
